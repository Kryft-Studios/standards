# Universal File Naming Style Guide For Kryft Studios

| Type | Case |
|--|--|
| Text File (.md, .txt, ...) | SCREAMING_SNAKE |
| Code File (.ts, .js, .cpp, ...) | lowerCamelCase |
| .json | lowerCamelCase |
| .yaml | lower-kebab-case |
| Ignore File (.gitignore, .npmignore, ...) | lowerCamelCase |
| Ecosystem Files (CMakeLists.txt, ...) | Not specified |

> ## File Extension Folders
> Any folder extending an file should follow the same case as the file
