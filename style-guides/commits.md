# Commit Style Guide for Kryft Studios

`
[1](...[2]): [3?]
`

1. **What happened in the commit:** A list of all things that happened in the commit seperated by a `-`. e.g. `feat-add`

2. **Files and their changes**: A list of all things that happened in each file seperated by a `-`. e.g. `@cosmos-js/core: Added Implementation, ...: ...`

3. ***[OPTIONAL]* Commit Overview**: Overview of all changes
